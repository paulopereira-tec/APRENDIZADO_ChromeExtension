WhatsApp / Without left sidebar

Esta extensão para o Google Chrome tem o objetivo de facilitar a integração de aplicações web com o WhatsApp Web.

Por vezes, é necessário fazer uso da API atual do WhatsApp se comunicar com alguns contatos e a barra lateral não deixa a nossa aplicação tão bonita quanto gostaríamos. A extensão WhatsApp / Without left sidebar some com a barra lateral do WhatsApp para deixar a aplicação mais bonita.

COMO USAR: Nos seus links, basta usar o parâmetro "source=wppExt". Se a extensão WhatsApp / Without left sidebar estiver instalada no cliente, será identificado e então abrirá um PopUp Window para essa tarefa.

EXEMPLO: https://api.whatsapp.com/send?phone=9999999999999&text=&source=wppExt&data=
